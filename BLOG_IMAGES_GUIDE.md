## Image Best Practices

### Recommended Specifications
- **Hero Images**: 1200x630px minimum (optimal for social sharing)
- **Supporting Images**: 800x600px minimum
- **File Format**: WebP (best compression) or JPEG (universal support)
- **Optimization**: Compress images before upload (use TinyPNG or similar)

### SEO Best Practices
- Add descriptive alt text to all images
- Include keywords naturally in alt text
- Use descriptive filenames (e.g., `california-notary-fees-calculator.jpg`)
- Lazy load images below the fold
- Provide proper image dimensions in HTML

### Accessibility
- Every image should have alt text
- Alt text should describe the image content
- Don't start alt text with "image of" or "picture of"
- Keep alt text under 125 characters when possible
